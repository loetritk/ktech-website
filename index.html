Microsoft Identity Manager

Identity & Account Management
ครบวงจรด้วย MIM

KTech ออกแบบและติดตั้ง Microsoft Identity Manager (MIM) สำหรับการ Synchronize ข้อมูล Identity จากระบบ HR, HRMS หรือ Database ใดๆ ไปยัง Active Directory, LDAP และระบบปลายทางอื่นๆ แบบอัตโนมัติและแม่นยำ

Identity Sync HR-Driven Provisioning Lifecycle Management Self-Service Portal

Authoritative Source → Target System

MIM ทำหน้าที่เป็น Identity Hub กลาง รับข้อมูลพนักงานจาก HRMS หรือ Database ต้นทาง แล้ว Sync สร้าง/แก้ไข/ปิด Account ใน Active Directory, Azure AD, LDAP และระบบต่างๆ โดยอัตโนมัติ ตาม Business Rules ขององค์กร

  • ✓ สร้าง AD Account อัตโนมัติเมื่อพนักงานใหม่เข้างาน (Joiner)
  • ✓ อัปเดตข้อมูลทันทีเมื่อพนักงานย้ายแผนกหรือเปลี่ยนตำแหน่ง (Mover)
  • ✓ ปิด/ระงับ Account และเพิกถอนสิทธิ์ทันทีเมื่อพนักงานออก (Leaver)
  • ✓ Password Self-Service Portal และ Group Management สำหรับผู้ใช้งาน
  • ✓ กำหนด Business Rules, Attribute Mapping และ Workflow ได้ยืดหยุ่น
